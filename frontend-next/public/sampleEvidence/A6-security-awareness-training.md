## 1. Purpose

This document describes the organization's security awareness and training program, implementing controls **A.6.3** (Security Awareness, Education and Training), **A.6.1** (Screening), and **A.6.8** (Information Security Event Reporting) of ISO/IEC 27001:2022 Annex A.

## 5. New Hire Security Onboarding

1. **Day 1**: NDA signing + Security Fundamentals module (mandatory before system access)
2. **Week 1**: Data Classification training + Acceptable Use Policy acknowledgment
3. **Month 1**: Role-specific security training completion
4. **Month 3**: First phishing simulation participation

## 6. Background Screening (A.6.1)

All candidates undergo pre-employment screening:
- Identity verification (government-issued ID)
- Criminal background check (past 7 years)
- Education and professional certification verification
- Reference checks (minimum 2 professional references)
- Credit check (for finance and executive roles only)

Screening is conducted by **[Third-party Provider]** with results retained for the duration of employment + 2 years.
